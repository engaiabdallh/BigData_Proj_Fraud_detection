# 🛡️ Fraud Detection System

An enterprise-grade fraud detection system powered by PySpark MLlib and Streamlit.

## 📋 Prerequisites

### Required Software
- **Python 3.9, 3.10, or 3.11** (not 3.12+)
- **Java 8 or 11** (JDK or JRE)
- **4GB+ RAM** recommended
- **Windows 10/11**, **macOS**, or **Linux**

### Installing Prerequisites

#### Windows
1. Install Python from [python.org](https://www.python.org/downloads/)
   - ✅ Check "Add Python to PATH"
2. Install Java from [Adoptium](https://adoptium.net/)
   - Download JDK 11 (LTS)
   - Run installer
3. Restart your computer

#### macOS
```bash
# Install Homebrew first (if not installed)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install Python and Java
brew install python@3.11 openjdk@11
